Test Case ID: TC-003

Title: Quantity field in cart cannot be edited

Preconditions:

User is logged in

At least one product is added to the cart

Steps to Reproduce:

Log in to the application

Add a product to the cart

Click on the cart icon

Try to click or change the quantity field

Expected Result:
The quantity field should be editable and allow the user to change the number of items.

Actual Result:
The quantity field cannot be clicked or edited.

Status: Failed
